# Online Food Order Simulation System

## About the Project
This is a simple C language project that simulates an online food ordering system.
The program shows a food menu, allows the user to select food items, asks for quantity,
and then calculates the total bill.

The user only selects the food items they want. Quantity is asked only for selected items.

---

## Features
- Menu displayed with food item numbers
- User selects only required food items
- Quantity input for selected items only
- Multiple food items can be ordered
- Total bill is calculated at the end

---

## Concepts Used
- C Language
- 1D Arrays
- Loops
- Functions
- If-else conditions
- Input and Output

---

## Menu
1. Burger   - Rs.100  
2. Pizza    - Rs.150  
3. Pasta    - Rs.200  
4. Fries    - Rs.80  
5. Sandwich - Rs.120  

Enter **0** to finish the order.

---

## Working of the Program
1. The program displays the food menu.
2. The user selects a food item number.
3. The program asks the quantity of the selected item.
4. The user can select more items if needed.
5. When the user enters 0, the total bill is displayed.
